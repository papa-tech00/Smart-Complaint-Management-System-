# AI Service

A Python-based AI/ML service project.

## Setup

1. Create virtual environment

```
python -m venv venv
```

2. Activate

Windows

```
venv\Scripts\activate
```

3. Install packages

```
pip install -r requirements.txt
```

## Tech Stack

- pandas
- numpy
- scikit-learn
- nltk
- flask
- joblib
- matplotlib
- seaborn
